## ⚡ Asosiy Xususiyatlari (Features)

*   **🔮 Shaffof va Silliq UI:** Glassmorphism effektlari hamda mukammal animatsiyalar (Framer Motion).
*   **⚡ Tezkor CRUD:** Vazifalarni bir lahzada qo‘shish, tahrirlash, o‘chirish va "bajarildi" deb belgilash.
*   **📊 Analitika va Progress Bar:** Kunlik ishlaringizning qancha qismi bajarilganini vizual tarzda kuzatish.
*   **📱 To‘liq Responsive:** Har qanday qurilmada (Smartfon, Planshet, Desktop) mukammal moslashuvchanlik.
*   **💾 Mahalliy Xotira (Persistence):** Sahifa yangilansa ham ma'lumotlar o‘chib ketmaydi (`LocalStorage` integratsiyasi).

## 🛠 Texnologik Stack

Loyihani professional va tezkor ishlashini ta'minlash uchun quyidagi texnologiyalardan foydalanildi:

| Qatlam | Texnologiya | Foydalanilgan Kutubxonalar / Qo‘shimchalar |
| :--- | :--- | :--- |
| **Frontend** | React.js / Next.js | Hooks (`useState`, `useEffect`, `useContext`) |
| **Styling** | Tailwind CSS | Custom Neon Gradients, Arbitrary values |
| **Animatsiyalar** | Framer Motion | Smooth Page Transitions & Micro-interactions |
| **Ikonkalar** | Lucide React | Minimalistik va chiziqli zamonaviy piktogrammalar |
