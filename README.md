## 🚀 Xususiyatlari (Features)

*   **Interaktiv interface:** Foydalanuvchi uchun qulay va responsive dizayn.
*   **Vazifalar nazorati:** Kunlik rejalarni qo‘shish, tahrirlash va o‘chirish (CRUD).
*   **Glassmorphism dizayn:** Zamonaviy UI/UX elementlari va silliq animatsiyalar.
*   **Statistika:** Bajarilgan ishlarni vizual tarzda kuzatish imkoniyati.

---

## 🛠 Texnologiyalar (Tech Stack)

Loyihani yaratishda quyidagi zamonaviy texnologiyalardan foydalanilgan:

| Texnologiya | Vazifasi |
| :--- | :--- |
| **React.js / Next.js** | Frontend arxitekturasi va komponentlar |
| **Tailwind CSS** | Tezkor va chiroyli stillar (Styling) |
| **Framer Motion** | Silliq va interaktiv animatsiyalar |
| **Node.js / LocalStorage** | Ma'lumotlarni saqlash va qayta ishlash |

## 💻 Loyihani kompyuterda ishga tushirish

Loyiha nusxasini lokal kompyuteringizda ishga tushirish uchun quyidagi qadamlarni bajaring:

1.  **Repository-ni klonlashtiring:**
```bash
    git clone [https://github.com/username/daywork.git](https://github.com/username/daywork.git)
    ```
2.  **Loyiha papkasiga o‘ting:**
```bash
    cd daywork
    ```
3.  **Kerakli paketlarni yuqlang (Install):**
```bash
    npm install
    # yoki
    yarn install
    ```
4.  **Loyihani ishga tushiring (Development server):**
```bash
    npm run dev
    # yoki
    yarn dev
    ```
